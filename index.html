Overview

attrchange is a simple jQuery function to add listener function to any HTML element on attribute change.

The plugin was written to answer a question posted on stack overflow

DEMO:

http://jsfiddle.net/zFVyv/10/

Browser Support Information:

DOMAttrModified - Supported in IE 9, FF and Opera

MutationObservers - is very new and it is supported in FF and Chrome 18.

onpropertychange - Only in IE
