Overview

attrchange is a simple jQuery function to add listener function to any HTML element on attribute change.

This plugin can be used to monitor the changes happening to a HTML element by some other code.

The plugin was originally written to answer a question posted on stack overflow

Basic Usage

Start using this plugin by following 2 simple steps.

Include attrchange.js to your web page 

<script src="https://raw.github.com/meetselva/attrchange/master/attrchange.js" type="text/javascript"></script>

Bind attrchange handler function to selected elements 

$(selector).attrchange(function () {
    //What you want to do when the selected elements attribute is added/updated/removed
});

$('#blocks .block').attrchange(function(attrName) {
  $('#logger').prepend('
- attribute ' + attrName + ' changed for ' + this.title + '
');
});
